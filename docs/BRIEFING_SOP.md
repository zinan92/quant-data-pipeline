> 版本: v1.0 | 生效日: 2026-02-04 | 维护: Wendy

## 概述

每个定时简报必须包含以下全部19个SKU模块。无一例外。

## 标准全量 SKU (每次必出)

### Section A — 基础数据

| SKU | 模块 | 数据源 | 说明 |
|-----|------|--------|------|
| `IDX-CORE` | A股主要指数 | Sina hq.sinajs.cn | 上证/深证/创业板/科创50：价格+涨跌幅+成交额 |
| `LIMIT-STATS` | 涨跌停统计 | AKShare (东方财富) | 涨停/跌停数量 + TOP3代表个股 |
| `BIG-TRADE` | 大笔买卖 | AKShare (东方财富) | 大笔买入/卖出数量 + 净差额 |
| `WATCH-MOVER` | 自选股异动 | Sina + 本地watchlist | 涨幅前5 + 跌幅前5 (348只自选股) |

### Section B — 新闻/情报

| SKU | 模块 | 数据源 | 说明 |
|-----|------|--------|------|
| `NEWS-CN` | 中文财经快讯 | 同花顺/财联社/新浪 | 最新5-8条快讯标题 |
| `KW-TRIGGER` | 关键词触发 | 快讯全文扫描 | AI/芯片/新能源/贵金属/政策 命中高亮 |
| `SOUTH-FLOW` | 南向资金 | AKShare (东方财富) | 港股通(沪)+港股通(深) 净买入额 |

### Section C — 概念资金流

| SKU | 模块 | 数据源 | 说明 |
|-----|------|--------|------|
| `FLOW-TOP20` | 概念资金流TOP20 | Tushare moneyflow_ths | ≥30亿门槛排名。**盘中不可用时用昨日参考** |
| `FLOW-COMPARE` | 资金流日度对比 | Tushare | 昨日vs今日，加速/减速/转正。**盘中不可用时标注** |
| `FLOW-TREND` | 资金流多日趋势 | Tushare | 3日/5日累计+方向箭头。加速流入/连续流入标记 |
| `FLOW-ROTATE` | 板块轮动检测 | Sina概念实时 + 快照对比 | 上半小时vs这半小时概念排名变化 |

### Section D — 行情复盘

| SKU | 模块 | 数据源 | 说明 |
|-----|------|--------|------|
| `INTRADAY-TABLE` | 盘中全程回顾 | 本地快照序列 | 每个cron时间点指数快照对比表格 |
| `PATTERN-ID` | 行情模式识别 | 快照分析 | V型反转/单边跌/震荡/分化 自动判断 |
| `MAINLINE` | 主线梳理 | Sina概念实时排名 | 今日领涨/领跌板块 + 盘面特征分析 |

### Section E — 信号/监控

| SKU | 模块 | 数据源 | 说明 |
|-----|------|--------|------|
| `MOMENTUM` | 动量信号 | 本地cron (每2分钟) | 上涨激增+K线形态检测。独立运行，简报中报告状态 |
| `SMART-ALERT` | 智能推送 | /api/news/smart-alerts/scan | 关键词规则匹配+涨停批量提醒 |

### Section F — 特殊时段 (叠加在全量之上)

| SKU | 模块 | 触发时间 | 说明 |
|-----|------|---------|------|
| `PRE-OPEN` | 盘前简报 | 09:20 | 隔夜美股收盘回顾 + 今日关注板块 + 经济日历事件 |
| `MID-REVIEW` | 午间复盘 | 11:30 | 上午走势总结文字 + 上午资金流对比 |
| `CLOSE-FULL` | 收盘总结 | 15:05 | 全天复盘 + 完整资金流(Tushare当日) + 5日趋势 + 全日走势表 |

## 数据源依赖

| 数据源 | 可用时间 | 备注 |
|--------|---------|------|
| Sina 实时行情 | 9:15-15:15 | 指数+个股报价 |
| AKShare 东方财富 | 9:30-15:00 | 涨跌停池、大笔买卖 |
| 同花顺/财联社快讯 | 全天 | 新闻流 |
| Tushare moneyflow | 收盘后 (~16:00) | 个股资金流→概念聚合 |
| Sina 概念板块 | 9:30-15:00 | 实时概念涨跌排名 |
| 本地快照 | 每次cron | 指数时间序列 |

### 盘中 Tushare 不可用时的处理
- FLOW-TOP20: 展示**昨日**排名作为参考
- FLOW-COMPARE: 标注"盘中不可用"
- FLOW-TREND: 展示截至昨日的趋势数据
- 收盘简报(15:05): 等Tushare更新后输出完整数据

## 存储

每次简报生成后必须保存到两个位置:

1. **wendy-tracker**: `wendy-tracker/public/data/briefings/YYYY-MM-DD.json`
   - 每天一个文件，追加每次简报
   - 格式: `{date, briefings: [{time, type, content, skus_included}]}`

2. **Notion**: `每日行情简报` 数据库 (待创建)
   - 每天一条记录
   - Properties: 日期, 时间, 类型(盘中/午间/收盘), 简报内容

## 执行脚本

```bash

cd ~/clawd/ashare && source .venv/bin/activate

python scripts/full_briefing.py --time 11:30 --midday

# 收盘简报
python scripts/full_briefing.py --time 15:05 --closing

# 自动判断模式 (cron推荐)
python scripts/full_briefing.py --auto
```

## 质量检查

每次简报输出前，检查:
- [ ] 19个SKU全部有输出 (即使某个标注"不可用")
- [ ] 指数数据是实时的 (非缓存)
- [ ] 快讯是最新的 (非重复)
- [ ] 关键词扫描覆盖最近30条新闻
- [ ] 已保存到 wendy-tracker
- [ ] 已保存到 Notion
